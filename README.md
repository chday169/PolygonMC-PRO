# PolygonMC-PRO
Implicit 3D Surface Engine for TurboWarp / Scratch  by Ching-Her Day (戴清河, chday169)  🌐 Overview  PolygonMC PRO is a TurboWarp / Scratch custom extension for rendering implicit 3D surfaces defined by:  f(x,y,z)=0  It converts mathematical functions into 3D polygon meshes in real-time using a lightweight polygon reconstruction method
